# Quantum Computing Specialization
This repository contains the code and materials for the [Quantum Computing Specialization course](https://apps.uc.pt/courses/PT/course/11701) at [University of Coimbra](https://www.uc.pt/). 

The specialization covers fundamental concepts of quantum computing, quantum algorithms, and practical implementations using Qiskit.

## Course Modules
The specialization is divided into the following modules:

1. [**Introduction to Quantum Information and Computing**](https://apps.uc.pt/courses/PT/unit/95867/24342/2025-2026?common_core=true&type=ram&id=11701): Basics of quantum mechanics and qubits.

2. [**Quantum Algorithms and Applications**](https://apps.uc.pt/courses/PT/unit/95868/24342/2025-2026?common_core=true&type=ram&id=11701): Study of key quantum algorithms such as Grover's and Shor's algorithms.
